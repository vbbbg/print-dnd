# 工具栏组件

## 技术栈

shadcn

## 功能需求

### 1. 撤回/重做

- 实现撤销和重做功能
- **历史记录策略**：仅在拖动操作开始时记录状态快照
- 每次拖动操作产生一个历史记录，而非拖动过程中的每次状态变化

### 2. 视图比例

- 默认缩放比例：100%
- 缩放调整：点击 + / - 按钮，每次增减 10%
- 不支持手动输入比例值（只读显示）

### 3. 恢复默认布局

- 一键恢复模板至初始默认布局
- 重置所有元素位置和区域大小

### 4. 打印预览

- 当前阶段：仅实现按钮 UI
- 功能占位，暂不实现具体预览逻辑

### 5. 另存为新模板

- 当前阶段：仅实现按钮 UI
- 功能占位，暂不实现具体保存逻辑
